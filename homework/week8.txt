Последняя домашка курса ООП

1. Написать приложение для управления компанией с помощью библиотеки Swing
	Приложение должно иметь следующий функционал
	
	- добавить сотрудника
	- удалить сотрудника
	- редактировать сотрудника
	- отобразить всех сотрудников
	* добавить фильтр при отображению сотрудников (сортировка списка по имени, по зп, по возрасту) 
	* сделать педжинейшн, т.е. отображать сотрудников порциями(по 5, 10, 20 на странице) - количество выбирает пользователь
	- при выходе из программы сохранять сотрудников в файл
	- при запуске программы загружать сотрудников из файла
	* сотрудник может иметь фотографию
	* искать сотрудника по ФИО
	- чтобы зайти в приложение нужно ввести логин и пароль
	* если пользователь не делает никаких действий в течении 5 минут, ему нужно ввести логин и пароль еще раз
